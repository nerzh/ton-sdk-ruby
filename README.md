# ton-sdk-ruby

Ruby SDK for interaction with TON (The Open Network) blockchain

## Installation

Install ton-sdk-ruby:

- `gem install ton-sdk-ruby`

## Example

```ruby
require 'ton-sdk-ruby'

class Ton
  include TonSdkRuby
  
  def main
    
  end
end

Ton.new.main
```
